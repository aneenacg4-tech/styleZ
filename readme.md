# Trend 2026 Webpage

## Overview

This project is a responsive landing page titled **"What's Trend in 2026"**, designed using **HTML**, **Tailwind CSS**, **Google Fonts**, and **Font Awesome**. It showcases trending topics with images, buttons for user interaction, and a clean modern layout optimized for both desktop and mobile devices.

---

## Features

* **Responsive Design**: Works on desktop and mobile screens with a flexible layout.
* **Custom Fonts**: Uses **Orbitron** for headings and **Poppins** for body text via Google Fonts.
* **Navigation Bar**: Includes a logo, menu items, and search functionality.
* **Hero Section**: Highlights trending content with images and descriptive text.
* **Buttons**: Call-to-action buttons such as "Email ID" and "Join Now".
* **Social Media Icons**: Integrates Font Awesome icons for social contact links.
* **Modern Styling**: Tailwind CSS used for quick styling and responsive design.

## How to Use

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in your web browser.
3. Replace placeholder images (`logo.png`, `Image.png`, etc.) with your own assets if desired.
4. Edit text content or styling by modifying the HTML and Tailwind classes.

## Notes

* The current layout is **optimized for large screens**; some adjustments may be needed for small devices.
* Tailwind CSS is loaded via CDN for quick prototyping. For production, consider a local setup for better performance.
* Ensure all images are located in the correct paths as referenced in the HTML.
