## Overview

This landing page highlights "what’s trending in 2026" with a visually engaging layout. It features a hero section with images, a navigation bar, call-to-action buttons, and social/contact icons. The design adapts for mobile, tablet, and desktop screens.

---

## Features

- Fully responsive layout for different screen sizes
- Navigation bar with hover effects
- Hero section with layered images for visual depth
- Bold typography using Google Fonts (Orbitron & Poppins)
- Interactive buttons (Email ID & Join Now)
- Social and contact icons using Font Awesome

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/2026-trends-landing-page.git
````

2. Navigate into the project folder:

   ```bash
   cd 2026-trends-landing-page
   ```

3. Open `index.html` in your browser:

   ```bash
   open index.html
   ```

## Usage

* Use the navigation menu to explore sections (HOME, MALE, FEMALE, FAQ, ABOUT).
* Hover over menu items to see underline and color effects.
* View the hero section with images that overlap for a layered effect.
* Click the buttons (`EMAIL ID`, `JOIN NOW`) for call-to-action (currently placeholder functionality).
* Social icons at the bottom allow quick access to contact methods.
