## Usage

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Ensure you have an internet connection to load Google Fonts, Tailwind CSS, and Font Awesome.

---

## Customization

* **Fonts:** Change in the `<head>` section by modifying the Google Fonts link and Tailwind config.
* **Colors & Styling:** Adjust Tailwind CSS classes in HTML elements for colors, padding, and layout.
* **Images:** Replace images in the `src` attributes with your own.
